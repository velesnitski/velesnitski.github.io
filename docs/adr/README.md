# Decision Records

A lightweight log of notable decisions for this site — mostly **positioning and content** choices (what to say, what to reveal, how to present), plus the occasional tech/design call.

Not full architecture ceremony — this is a static landing page. The point is to capture the **"why"** so past decisions don't get silently re-litigated.

## Format

Each record: one file, `NNNN-short-kebab-title.md`, with:

- **Date** · **Status** (Proposed / Accepted / Superseded)
- **Context** — what prompted it
- **Decision** — what was chosen
- **Consequences** — what follows, and what to revisit later

Number sequentially (`0001`, `0002`, …). Supersede rather than delete: mark the old one `Superseded by NNNN` and link.
